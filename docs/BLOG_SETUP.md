# Инструкция по настройке блога

## 📁 Структура папок

```
src/content/blog/
├── post-1.md          # Русский пост
├── post-1.en.md       # Английская версия (опционально)
├── post-2.md          # Русский пост
└── post-2.en.md       # Английская версия (опционально)
```

## 🖼️ Как добавить картинку (обложку)

### Вариант 1: Положить в папку `public/` (рекомендуется)

1. Положи картинку в папку `public/` в корне проекта:
   ```
   public/
   ├── blog-images/
   │   ├── post-1-cover.webp
   │   └── post-2-cover.webp
   ```

2. В frontmatter поста укажи путь:
   ```yaml
   ---
   title: "Название поста"
   description: "Описание поста"
   date: 2025-03-22
   lang: "ru"
   image:
     url: "/blog-images/post-1-cover.webp"
     alt: "Описание картинки для SEO"
   ---
   ```

### Вариант 2: Положить в папку `src/content/blog/`

1. Положи картинку в ту же папку, где лежит пост:
   ```
   src/content/blog/
   ├── post-1.md
   └── post-1-cover.webp
   ```

2. В frontmatter поста укажи путь:
   ```yaml
   ---
   title: "Название поста"
   description: "Описание поста"
   date: 2025-03-22
   lang: "ru"
   image:
     url: "./post-1-cover.webp"
     alt: "Описание картинки для SEO"
   ---
   ```

## 📝 Как создать пост

### Шаг 1: Создай файл

Создай файл в папке `src/content/blog/`:
- Для русского поста: `my-post.md`
- Для английской версии: `my-post.en.md`

### Шаг 2: Заполни frontmatter

```yaml
---
title: "Название поста"
description: "Краткое описание для SEO и превью"
date: 2025-03-22
lang: "ru"  # или "en" для английской версии
draft: false  # true, если пост не готов к публикации
tags: ["AI", "Crypto", "DeFi"]  # опционально
series: "Название серии"  # опционально, для серии постов
image:
  url: "/blog-images/my-post-cover.webp"
  alt: "Описание картинки для SEO"
---
```

### Шаг 3: Напиши контент

```markdown
---
title: "Как я создал AI-агента для крипто-арбитража"
description: "История создания Telegram-бота для Polymarket"
date: 2025-03-22
lang: "ru"
image:
  url: "/blog-images/polymarket-bot-cover.webp"
  alt: "Скриншот интерфейса Telegram-бота"
---

## Введение

Здесь твой контент...

## Основная часть

Текст поста...

## Заключение

Выводы...
```

## 🌍 Как настроить язык поста

### Русский пост

```yaml
---
title: "Название поста"
description: "Описание поста"
date: 2025-03-22
lang: "ru"
---
```

### Английский пост

```yaml
---
title: "Post Title"
description: "Post description"
date: 2025-03-22
lang: "en"
---
```

### Как работает фильтрация

- Если `lang: "ru"` — пост показывается только на русском языке
- Если `lang: "en"` — пост показывается только на английском языке
- Если `lang` не указан — пост показывается на обоих языках

### Пример пары постов

**Русский пост** (`src/content/blog/my-post.md`):
```yaml
---
title: "Как я создал AI-агента"
description: "История создания"
date: 2025-03-22
lang: "ru"
---
```

**Английская версия** (`src/content/blog/my-post.en.md`):
```yaml
---
title: "How I Created an AI Agent"
description: "Creation story"
date: 2025-03-22
lang: "en"
---
```

## 🎯 Рекомендации

### Форматы картинок
- Используй `.webp` для лучшей производительности
- Оптимальный размер: 1200x630px (для соцсетей)
- Максимальный размер файла: 500KB

### SEO
- `title` — должен быть уникальным и содержать ключевые слова
- `description` — до 160 символов, краткое описание
- `image.alt` — описание картинки для поисковиков

### Названия файлов
- Используй только латинские буквы, цифры и дефисы
- Пример: `my-post.md`, `ai-agents-guide.md`
- Не используй пробелы и спецсимволы

### Даты
- Формат: `YYYY-MM-DD`
- Пример: `2025-03-22`

## 🚀 После создания поста

1. Перезапусти dev сервер (если он запущен)
2. Проверь пост на странице `/blog`
3. Проверь отдельную страницу поста `/blog/my-post`
4. Проверь переключение языка — пост должен фильтроваться
