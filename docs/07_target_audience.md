# Целевая аудитория

## Кто клиенты

1. **Финтех-стартапы**
   - Нужны AI-агенты и автоматизация
   - Ищут быстрый MVP (4–14 дней)

2. **Крипто-проекты**
   - Нужны onchain-аналитика и DeFi-инструменты
   - Арбитраж-боты, автоматизация ликвидности

3. **Инвесторы и трейдеры в DeFi**
   - Нужны инструменты для анализа и автоматизации
   - Поиск пулов ликвидности, мониторинг позиций

4. **Блогеры с финансовой аудиторией**
   - Теряют доход из-за угрозы блокировки Telegram
   - Нужен собственный продукт под свою аудиторию
   - Есть трафик, нет технической команды
