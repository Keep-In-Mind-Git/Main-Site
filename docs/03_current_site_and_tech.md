## Технологический стек проекта

### Фреймворк
- **Astro** — современный фреймворк для статических сайтов

### Текущая структура проекта
```
src/
├── components/          # Компоненты UI
│   ├── About.astro
│   ├── Card.astro
│   ├── Contact.astro
│   ├── Footer.astro
│   ├── Head.astro
│   ├── Header.astro
│   ├── Hero.astro
│   ├── Posts.astro
│   └── Projects.astro
├── content/            # Контент (Markdown)
│   ├── blog/           # Блог
│   └── projects/       # Проекты/Кейсы
├── layouts/            # Layouts
├── pages/              # Страницы
└── styles/             # Стили
```
