## Контекст

SMM-агентство теряло клиентов из-за конкуренции с ChatGPT. Их текущий автопостинг на 20 аккаунтов стоил 6 500 руб/мес и регулярно давал сбои. Нужна была AI-трансформация процессов.

## Задача

Заменить сломанный автопостинг и добавить AI-мониторинг трендов для опережения конкурентов.

## Что сделал

- Миграция 20 аккаунтов на Late.dev API через n8n
- Notion Database как центр управления контентом (Notion → n8n → Late.dev → Telegram/X/LinkedIn)
- Telegram-алерты при любых ошибках постинга
- AI Trend Tracker: парсинг 10 источников раз в день, GPT-кластеризация тем, алерт если тема в >50% источников

## Результат

> Трендвотчер используется ежедневно — агентство публикует контент на опережение и набирает больше охватов. Автопостинг работает без сбоев.

## Технологии

- Notion API
- Late.dev API
- OpenAI GPT API
- Telegram Bot API
