## Context

DeFi traders spent 15-20 minutes manually searching for suitable liquidity pools on Meteora for a specific token. There was no tool for quick searching.

## Task

A Chrome extension that automatically finds liquidity pools for a selected token.

## What I did

- Integration with GmGn – pulls the token's ticker and address with one click
- Automatic search for all available pools on Meteora
- Filtering by farming parameters

## Result

> Finding the right pool takes seconds instead of 20 minutes. A leading DeFi blogger published the extension on his channel.

## Technologies

- JavaScript
- Chrome Extension API
- Meteora API
- GmGn API
