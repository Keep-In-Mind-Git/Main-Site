## Context

Coffee shops and restaurants are losing loyal customers due to the lack of proper analytics and loyalty programs. Existing solutions are expensive and complex for small establishments.

## Challenge

MVP of a loyalty system with analytics integrated into existing POS systems.

## What I did

- A fully functional product in 2 months — from concept to first client
- Integration with iiko (a popular HoReCa POS system)
- Web admin panel with analytics
- Telegram bot for guests
- Development entirely using AI tools — hence the speed

## Result

> The first paying client has been connected. The product is in active iteration. We are measuring metrics before and after implementation.

## Technologies

- Python
- iiko API
- Telegram Bot API
- React
- PostgreSQL
