## Контекст

DeFi-трейдеры тратили 15–20 минут на ручной поиск подходящих пулов ликвидности на Meteora под конкретный токен. Инструмента для быстрого поиска не существовало.

## Задача

Расширение для Chrome, которое автоматически находит пулы ликвидности под выбранный токен.

## Что сделал

- Интеграция с GmGn — тянет тикер и адрес токена одним кликом
- Автоматический поиск всех доступных пулов на Meteora
- Фильтрация по параметрам фарминга

## Результат

> Поиск нужного пула — секунды вместо 20 минут. Расширение опубликовал крупный DeFi-блогер у себя в канале.

## Технологии

- JavaScript
- Chrome Extension API
- Meteora API
- GmGn API
